# Icon Placeholders

The `manifest.json` references `.png` icon files:

- `icon-16.png`
- `icon-48.png`
- `icon-128.png`

**Before loading the extension into Chrome**, convert the SVG placeholder or provide properly sized PNG files here.

You can generate PNGs from the SVG using any tool (e.g., Inkscape, Figma, or an online converter).
